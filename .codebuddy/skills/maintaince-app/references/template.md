# 提交描述模板模块

## 功能说明

git提交描述模板。

## AI 标识要求

**所有通过 AI Agent 发起的 提交 必须在描述开头添加 AI 标识：**

- **标识格式**：使用引用块 (>) 突出显示，包含 🤖 emoji
- **位置要求**：必须在描述的最开头，第一行就是 AI 标识

## 标准模板

```markdown
> 🤖 本 提交 由 AI Agent 自动发起
>
> ⚠️ 请仔细审查代码变更，AI 生成的代码可能存在问题

## 变更描述
[简要描述本次变更的目的和内容]

## 变更内容
- [ ] [具体变更项 1]
- [ ] [具体变更项 2]
- [ ] [具体变更项 3]

## 测试情况
- [ ] 单元测试通过
- [ ] 集成测试通过
- [ ] [其他测试]


🤖 Generated with maintaince-app(SKILL)
```

## 工蜂平台标题格式

工蜂平台的 标题和描述使用特定前缀：

- 标题格式：`{type}({scope}): {description}`
- 描述格式：`[上述模板内容]`
- 描述正文语言：**中文为主，英文仅用于必要的专有名词或技术术语**

**示例：**
```
标题：feat(player): 添加玩家等级系统
详情：> 🤖 本提交由 AI Agent 自动发起
> ...
```
